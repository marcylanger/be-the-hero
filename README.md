# be-the-hero
O projeto foi desenvolvido com base no conteúdo disponibilizada na Semana OmniStack 11.0 e conta com a seguinte estrutura e respectivas tecnologias:
* BackEnd: Node.js
* FrontEnd: React
* Mobile: React Native
